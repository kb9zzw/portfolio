    Introduction

    This web survey application was created to help scientists collect information about climate vulnerability and resilience along the north shore of Lake Superior. This project involved two aspects:

    • Create compelling user interface that can be used to collect climate data from the people who are familiar with the region.
    • Provide a means to analyze the user input to find areas that might be impacted

    One of the issues when implementing open-source GIS tools is interoperability with proprietary tools, such as ArcGIS. To help with this, the application also implements a number of interoperability features to allow data to be exported from the open-source implementation into formats that can be readily consumed by ArcGIS and other tools. This includes implementation of shapefile and KML export capabilities. Web services are available that can export the data as OGC WFS and GeoJSON.

    Technologies

    User Interface Leaflet, JQuery, JavaScript, Bootstrap
    Application Apache, Python, GeoDjango
    Database PostgreSQL, PostGIS
    System Ubuntu

    This project demonstrates what is possible using open-source GIS technologies. The entire application, from the operating system to the user interface, is implemented using open source tools and frameworks.
