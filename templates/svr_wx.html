    Introduction

    This project was created as a final web mapping demostration project for GIS 515 in North Carolina State University's MGIST program. The goal of this project was to explore cartographic representation of GIS data using web technologies. For this project, I chose to explore Mapbox as the primary technology used to highlight the 2011 severe weather season. Mapbox was chosen because it provides a rich user experience for users to explore spatial data in a dynamic and interactive way.

    User Interface Mapbox Javascript API, CSS
    Application & Tools Mapbox, TileMill, ArcGIS

    Raw data for this project was processed in ArcGIS, then converted to interactive layers using TileMill for loading into Mapbox. Mapbox tiles were added to the web site using the Mapbox Javascript API.
