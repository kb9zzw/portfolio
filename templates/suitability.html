    Introduction

    Suitability analysis involves combining data an attributes to find the most favorable location for an event. The example below shows how ArcGIS can be used to find the most favorable bear habitat locations in the Smoky Mountain National Park. The general process involved creating reclassified raster datasets for each parameter ranking the parameter from least favorable (1) to most favorable (3).

    The following parameters were used for this analysis:

    • Vegetation
    • Slope
    • Distance to Trails
    • Distance to Streams
    • Distance to Roads

    Suitability Model

    ArcGIS Model Builder was used in this case to analyze raw data and create reclassified raster data based on selection criteria..
