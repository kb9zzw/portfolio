    Introduction

    Regional Snowfall Index (RSI) is a method developed by NOAA to quantify snow storms based on snowfall totals and impact to populations. It bears some resemblance to other storm indices, such as the Saffir-Simpson scale for hurricanes and the Fujita scale for tornadoes. But, since snowfall events tend to be region and regional populations vary in their abilities to cope with snowfall, the index is calculated on a regional basis rather than a nation-wide basis.

    As part of my studies in the MGIST program, I developed two versions of the RSI tool. The first one is developed in Python for use within ArcGIS and is an adaption of the current RSI tool used operationally at NOAA. The second was developed in Python using GRASS as an experiment in data processing using open-source GIS tools.
