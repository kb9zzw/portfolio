Introduction

It is important when designing maps to keep in mind the audience and medium. Different media lend themselves to different graphical representations. For example, in many scientific journals, space is at a premium and color may not be available. This stands in contrast to modern web maps which make liberal use of color and add tremendous functionality.

Below are three examples of maps designed for different media.

Print Maps

This particular image compares the forecasted versus actual path of Hurricane Sandy.

The goal of this image was to create a graphic constrained by size and color, such as might be found in a scientific journal. Shades of gray and symbols are used to highlight map features.

Web Maps

This map shows hail events for 2011 by county.

Unlike static images, which are confined by size, web maps can provide a much more interactive experience.

Modern web maps can be panned and zoomed to provide the illusion of the entire world in a small box. This map was created in JavaScript using MapBox. The underlying data started off in ArcGIS Desktop and was converted from Shapefile to Mapbox Tile Layer using TileMill.
